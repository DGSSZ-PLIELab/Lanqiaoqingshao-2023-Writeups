**公式**： $num=d0​×100+d1​×101+d2​×102+...+dn​×10n$

**算法**

>1.`igit = temp % 10`：使用取模运算获取 `temp` 的个位数字
>
>*eg. `temp = 1634` -> `1634 % 10 = 4`*

>2.`temp /= 10`：整数除法，去掉已经处理的个位数
>
>*eg. $1634 / 10 = 163$*

例如如下代码，实质就是重复上述算法过程

```C++
while (temp) { // 等价于while(temp != 0)
    int digit = temp % 10;
    sum += pow4[digit];
    temp /= 10;
}
```
